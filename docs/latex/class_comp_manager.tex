\subsection{Подробное описание}
Менеджер компонентов. Класс нужен для более удобного управления составляющими механизма. Пользователю можно не объявлять все переменные самому, достаточно вызвать нужные методы этого класса.

В целом, класс позволяет добавлять:
\begin{DoxyItemize}
\item именованые сегменты
\item именованые коннекторы (с указанием имени сегмента, на который нужно добавить коннектор)
\item именованые кинематические пары (с указанием имён коннекторов, с которыми нужно связать КП)
\item именованые силы (с указанием коннектора, который будет являться точкой приложения силы)
\end{DoxyItemize}
