Подробное описание

Класс для загрузки механизмов из xml-файлов.

При этом для парсинга xml-файла используется библиотека TinyXML.

Методы

void XMLLoader::Load ( string  filename  )  [static]

Загружает механизм из файла с именем filename.

При этом вызывая нужные методы менеджера компонентов *c и менеджера групп *g

Аргументы:
filename Имя загружаемого xml-файла
