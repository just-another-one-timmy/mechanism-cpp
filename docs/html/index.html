TMM Project

Программа для анализа плоских механизмов Программа позволяет:

  • Создавать прототип механизма на компьютере для моделирования его работы
  • Автоматически разбивать механизма на группы Ассура

Ограничения:

  • Группы Ассура класса не выше 2
Предупреждения:
АМ = ассоциативный массив (stl map)
КП = Кинмеатическая пара
Необходимо сделать:

Переделать визуализатор

Вынести TinyXML в отдельную библиотеку

Сделать человеческое главное окно (переделать черновик на чистовик)

Ошибка:

Не все механизмы работают

Режим SM_HARD_STEPS_CNT не работает
