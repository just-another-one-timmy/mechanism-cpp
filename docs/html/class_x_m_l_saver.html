Подробное описание

Класс для сохранения механизма в файл xml.

После этого используя класс XMLLoader можно загрузить механизм из сохраненного файла. Класс является статическим.

Методы

void XMLSaver::Save ( string  filename  )  [static]

Сохраняет механизм в файл.

Сохраняет сначала cm в файл (информация о звеньях, КП, коннекторах), а затем gm (информация о группах)

Аргументы:
filename Имя файла, в который будет сохранен механизм.
Ошибка:
Пока что нет возможности передавать NULL вторым параметром, чтобы группы не сохранялись. А оно вообще надо?
void XMLSaver::SetManagers ( CompManager *  c,
GroupManager *  g 
) [static]

Установить менеджеры.

Устанавливает указатели на менеджер компонентов и менеджер групп, которые и составляют механизм.

Аргументы:
c,g Указатели на менеджеры
Ошибка:

Архитектурное решение может быть неверным. В частности, нет возможности работать в многопоточной среде. Вариант решения - создание не статического класса XMLSaver, но теряется удобство пользования.

Нет проверки, совпадают ли CompManager *c и g->cm
